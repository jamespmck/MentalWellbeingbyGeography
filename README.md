# MentalWellbeingByGeography

A Good Measure data and evidence project building a geography-scoped predictor universe for mental wellbeing, service context and community need analysis in Australia.

This repository tracks project code, documentation and lightweight audit artefacts. Raw data, processed data, logs, archives and large binary files are excluded by default.

## Project structure

- scripts/ project acquisition, integration, audit and modelling scripts
- docs/ methodology notes, source registers and data dictionaries
- outputs/audits/ lightweight audit outputs used to document processing decisions

## Data note

Raw and processed datasets are not committed to this repository. Rebuilds should use the documented acquisition and processing scripts.
